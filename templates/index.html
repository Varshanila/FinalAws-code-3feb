AWS AI-Driven Personalized Marketing Platform

This project focuses on building an intelligent marketing platform using Artificial Intelligence concepts and Amazon Web Services (AWS). The system delivers personalized marketing campaigns by analyzing user behavior, preferences, and interaction patterns, helping businesses improve engagement and conversion rates.

AWS Services Used

Amazon EC2

Amazon EC2 is used to deploy and host the Flask-based web application. It provides scalable and reliable computing power for running the platform.

Amazon DynamoDB

DynamoDB is used as a NoSQL database to store user details, login information, and marketing campaign data with high performance and scalability.

Amazon SNS

Amazon SNS is used to send email notifications and alerts for user registrations, login activities, and campaign updates.

AWS IAM

AWS IAM manages secure access, roles, and permissions, ensuring that only authorized users and services can access AWS resources.
